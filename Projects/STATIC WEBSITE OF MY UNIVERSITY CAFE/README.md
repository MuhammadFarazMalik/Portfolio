## Summary
The Seven Eleven Cafe website, developed for United Cafe at UET Taxila, is a responsive static site built with HTML and CSS, featuring a hero banner, navigation bar, menu (Breakfast, Lunch, Snacks, Dinner), chef profiles, about section, and contact form. Using flexbox, media queries, and the Poppins font, it ensures accessibility across devices, tested on Chrome, Firefox, and Safari.
